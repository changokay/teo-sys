📦 수출 PO 관리

⚙️ Google Apps Script 연동이 필요합니다

1. Google 스프레드시트를 새로 만들고 → 확장 프로그램 > Apps Script 열기
2. po-script.gs 내용을 붙여넣고 저장
3. 배포 > 새 배포 > 웹 앱 → 액세스: 모든 사용자 → 배포
4. 발급된 웹 앱 URL을 아래 APPS_SCRIPT_URL에 입력 후 저장
5. Apps Script에서 setupTrigger() 한 번 실행 (1시간마다 자동 수집)
